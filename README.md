# vcloud

### 模块说明
vcloud-server 微服务<br>
vcloud-zuul 网关集群<br>
vcloud-core 核心类<br>
vcloud-admin 权限管理模块<br>
vcloud-test 示例代码模块<br>
vcloud-app 前端<br>

### 框架
springboot2.1.3<br>
spring5<br>
mybatis<br>
spring-security<br>
jwt<br>
swagger2<br>
junit4<br>

mysql5.6<br>
maven3.6.0<br>

### 运行环境说明
